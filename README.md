# etoshon.github.io
My Portfolio Website

This is a minimalist portfolio website showcasing my work in music, animation, photography, and product design.

## Pages
- **Home**: Welcome page with navigation
- **About**: Description about myself
- **Music**: Videos of my performances
- **Animation**: My animation work
- **Photography**: Selection of photos
- **Product**: Gallery of designs and products I have helped build

## Setup
This is a static HTML/CSS website. To view locally, open `index.html` in a browser.

For GitHub Pages, push to the `main` branch.

## Assets
- Images: Place in `images/` folder
- Videos: Place in `videos/` folder
